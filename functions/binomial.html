binomal( n , k )

The binomial coefficient in SageMath. Defined for integer arguments by

\[ \binom{ n }{ k } = \frac{ n! }{ (n-k)! \: k! }\]

and for one noninteger argument by

\[ \binom{ x }{ k } = \frac{ x(x-1) \cdots (x-k+1) }{ k! }\]
