chebyshev_T( n , z )

The Chebyshev polynomial of the first kind of z in SageMath. A solution of the differential equation

\[ ( 1-z^2 ) \frac{ d^2 f }{ dz^2 } -z \frac{ d f }{ dz } + n^2 f = 0 \]
