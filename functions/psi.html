psi( [ times , ] z )

The digamma function of z in SageMath. Defined by

\[ \psi(z) = \frac{ d }{ dz } \ln \Gamma(z) = \frac{ \Gamma(z)' }{ \Gamma(z) } \]

The optional times argument is used for multiple derivatives.
