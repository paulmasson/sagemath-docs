assume( statement )

An assumption in SageMath used in certain evaluations, mainly by Maxima but also Pynac.

Acceptable features used by Maxima:

integer
noninteger
even
odd
rational
irrational
real
imaginary
complex

analytic
increasing
decreasing
oddfun
evenfun
posfun
constant
commutative
lassociative
rassociative
symmetric
antisymmetric
integervalued

Examples:

var('n')
assume( n != -1 )
show( integrate( x^n, x ) )
