show( object [ , object , ... ] )

Default nicely formatted output in SageMath. Can take multiple arguments of the same type.

To display an unevaluated function append ,hold=True to the argument

Examples:

show( sin(2*pi,hold=True), '=', sin(2*pi) )

show( plot( sin(x), x, 0, 4*pi, figsize=[4.5,2], color=(0,.5,1) ) )
